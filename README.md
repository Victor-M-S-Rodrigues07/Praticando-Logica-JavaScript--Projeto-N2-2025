<h3> Status das Aplicações: Concluídas ✔ </h3>

## <h2> 📝 Descrição do Projeto </h2>

Este repositório contém 5 (cinco) projetos web desenvolvidos para o curso "Lógica de Programação: Praticando com Desafios", da Alura. Cada projeto apresenta peculiaridades que são requisitadas no mercado de trabalho, abragendo desde E-commerce até telas de cinema e aluguel de jogos. <br>

Foi disponibilizado pelos professores o HTML e o CSS das aplicações, cabendo somente a implementação da lógica, usando o JavaScript, para manipular elementos da página a fim de atingir o objetivo proposto. 
<br>

## <h2> 🎯 Objetivos da Aprendizagem </h2>

* Fortalecer os conceitos, sintaxe e semântico da linguagem JavaScript;
* Desenvolver habilidades de manipulação de elementos da página via lógica;
* Alterar classes CSS por condicionais e laços de repetição;
* Praticar a lógica com projetos reais do dia a dia. <br>

## <h2 align = "center"> Projeto Nº 1: 🔢 Sorteador de Números - Site para Realizar Múltiplos Sorteios </h2> <br>

<div align = "center">

<img src = ".\00-imagens-README\sorteador-numeros.png" width = 700px alt = "Imagem do site Sorteador de Números">

</div>

### <h3> 📖 Descrição do Projeto </h3>

Esse projeto trata de um sorteador de números no qual o usuário insere a quantidade desejada, o número mínimo e o número máximo que irão sair.

### <h3> ⚙ Funcionalidades </h3>

* Campo para adicionar a quantidade de números a serem sorteados;
* Possibilidade de controlar o intervalo entre os números;
* Botão para reiniciar e limpar os campos sem a necessidade de atualizar a página.
<br>

## <h2 align = "center"> Projeto Nº 2: 🧩 Alugames - Plataforma de Aluguel de Boardgames </h2> <br>

<div align = "center">

<img src = ".\00-imagens-README\alugames.png" width = 700px alt = "Imagem do site Alugames">

</div>

### <h3> 📖 Descrição do Projeto </h3>

O projeto ilustra um site de aluguel de boardgames (jogos de tabuleiro digitais), onde o usuário póde escolher qual dos jogos disponíveis ele gostaria de alugar ou devolver. Foi utilizadas as técnicas de manipulação do HTML e ativação ou desativação de classes CSS através do JavaScript para criar o efeito de mudança na cor do botão e de seu texto interior.

### <h3> ⚙ Funcionalidades </h3>

* Uso do método "getElementById()" para acessar elementos HTML pelo JS;
* Mensagens de confirmação caso o usuário queira alugar um jogo (Validação);
* Mudança da classe de elementos usando o método "ClassList()";
* Contador global do número de aluguéis realizados. 
<br>

## <h2 align = "center"> Projeto Nº 3: 🛒 Carrinho de Compras - Seleção de Produtos para um E-commerce </h2> <br>

<div align = "center">

<img src = ".\00-imagens-README\carrinho-compras.png" width = 700px alt = "Imagem do site Carrinho de Compras Tecnológicas">

</div>

### <h3> 📖 Descrição do Projeto </h3>

A aplicação retrata uma aba de e-commerce, na qual o usuário pode escolher um produto, adicionar a quantidade que deseja comprar e adicionar ao carrinho.

### <h3> ⚙ Funcionalidades </h3>

* Opções de seleção e de produtos para o usuário escolher;
* Uso de campos para o usuário informar a quantidade do produto escolhido;
* Cálculos e impressão de preços e do resultado;
* Acesso aos valores escritos no HTML e lógica de botões do site.
<br>

## <h2 align = "center"> Projeto Nº 4 : 🎞 Ingresso Online - Site para Compra e Reserva de Assentos em um Cinema</h2> <br>

<div align = "center">

<img src = ".\00-imagens-README\ingresso.png" width = 700px alt = "Imagem do site de Compra de Ingresso Online com o painél mostrando a quantidade de cadeiras restante em cada setor">

</div>

### <h3> 📖 Descrição do Projeto </h3>

A aplicação retrata um site de compra de ingressos. O usuário pode escolher a quantidade de assentos que deseja reserva e, depois de escolher o setor da poltrona, será decrementado do número total.

### <h3> ⚙ Funcionalidades </h3>

* Opção para escolher o local da poltrona;
* Espaço para o usuário informar a quantidade de poltronas;
* Painel com a quantidade restante.
<br>

## <h2 align = "center"> Projeto Nº 5: 🎁 Amigo Secreto - Sorteador de Nomes para um Amigo Secreto </h2> <br>

<div align = "center">

<img src = ".\00-imagens-README\amigo-secreto.png" width = 700px alt = "Imagem do site Amigo Secreto">

</div>

### <h3> 📖 Descrição do Projeto </h3>

Esse projeto é um sorteador de amigos para uma brincadeira de Amigo Secreto, onde uma pessoa tira o nome da outra para dar o presente no dia marcado. O usuário digita os nomes dos participantes (no mínimo quatro (4)) e aperta em sortear, recebendo a lista de quem tirou quem. 

### <h3> ⚙ Funcionalidades </h3>

* Utilização dos Vetores para armazenar nomes;
* Validações em caso de o usuário inserir somento um (1) participante, sorteio sem nenhuma pessoa ou nomes repetidos na lista;
* "textContent" e "innerHTML" para inclusão da lista no HTML da página;
* Funções Math.random das documentações para a geração de números aleatórios.
